# AI Challenge - Claude Documentation

## Project Overview

This is a Kotlin-based command-line AI chatbot application that enables interactive communication with multiple AI model providers through a unified CLI interface.

## Quick Facts

- **Language**: Kotlin 2.2.0
- **Build System**: Gradle 8.14
- **Target Platform**: JVM (Java 17)
- **Main Dependencies**: Ktor Client 3.0.2, Kotlinx Serialization 1.7.3, Coroutines 1.9.0
- **AI Providers**: Perplexity AI, HuggingFace Router
- **Interface**: Command-line (CLI)
- **Key Features**: Multi-client support, conversation history, metrics tracking, command chaining

## Repository Information

- **Current Branch**: ai-chellenge-day-7
- **Recent Development**: Progressive feature additions through daily challenges including token tracking, time measurement, terminal colors, and command queue functionality
